# Landing UF - Espacio Multimedia

Este proyecto es una pagina web tipo landing enfocada en contenido multimedia sobre coches.

## Que contiene la pagina

- Galeria de imagenes en carrusel (Aston Martin, Ferrari y Red Bull).
- Reproductor de video (`cars.mp4`) con controles personalizados.
- Dos reproductores de audio (`.ogg` y `.wav`) con controles personalizados.
- Formulario de contacto con validacion en el navegador.
- Mensajes visuales con SweetAlert2 al enviar el formulario.
- Bloque de enlaces a redes sociales (Instagram, X, LinkedIn y GitHub).

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (vanilla)
- Bootstrap 5 (CDN)
- SweetAlert2 (CDN)

## Estructura del proyecto

- `index.html`: estructura principal de la pagina.
- `styles.css`: estilos visuales y responsive.
- `player.js`: logica de controles multimedia y formulario.
- `audio/`: archivos de audio usados en la web.
- Imagenes y video en la raiz del proyecto (`.jpg`, `.mp4`).

## Funcionalidades destacadas

- Controles manuales para play, pausa, avanzar, retroceder, volumen, velocidad y mute en audio/video.
- Diseno responsive para escritorio, tablet y movil.
- Validacion simple del formulario antes de enviar.
